## GENERIK

Generik membuat class ataupun method yang fleksibel terhadap tipe data serta dapat memungkinkan penggunaan ulang suatu kelas. Tipe data belum ditentukan saat program ditulis dan baru akan ditentukan saat pembuatan objek atau pemanggilan method. Jadi dapat digunakan oleh tipe data berbeda dan tidak perlu menulis lagi kode yang sama berulang kali.
